Spida Phobia is a 3D Virtual Reality application made for a school project at the Université du Québec à Chicoutimi (UQAC). The course is the following : 3D interaction and virtual reality (8INF849).

This application is design to help people scared by spiders. The player can go through differents levels that involve different type of spiders. Visual of spiders can change (color, behavior, size). Each level contains a unique environnement. Finally, haptic vibration are used to emphasis the immersion.

Project Goals

  • Apply basics principles of virtual reality with tools used in industry.
  • Learn the basis of Unreal Development Kit (UDK).
